# Landing Page Sederhana

Ini adalah contoh landing page sederhana yang dibuat menggunakan HTML dan CSS. Halaman ini berfungsi sebagai template dasar yang dapat disesuaikan untuk berbagai tujuan, seperti promosi produk, layanan, atau kampanye.

## Fitur

- Navigasi header yang responsif.
- Hero section dengan tombol call-to-action.
- Seksi fitur utama untuk menampilkan keunggulan produk atau layanan.
- Footer dengan informasi hak cipta.

## Struktur Proyek

## Cara Menggunakan

1. **Clone repositori ini** atau unduh file secara manual:
    ```bash
    git clone https://github.com/username/landing-page.git
    ```

2. **Buka file `index.html` di browser** untuk melihat tampilan landing page.

3. Jika ingin mengubah tampilan:
    - Edit file `index.html` untuk mengubah konten dan struktur HTML.
    - Edit file `styles.css` untuk menyesuaikan desain dan gaya halaman.

## Penjelasan Kode

### `index.html`
- Halaman ini berisi struktur dasar HTML, termasuk bagian header, hero section, fitur utama, dan footer.
- Navigasi menu berada di dalam elemen `<header>` yang mencakup tautan ke bagian-bagian berbeda dari halaman.
- Hero section berfungsi sebagai area pengantar utama dengan teks sambutan dan tombol Call-To-Action (CTA).
- Bagian fitur menampilkan informasi tentang produk atau layanan yang dapat disesuaikan.
  
### `styles.css`
- Mengatur desain halaman dengan menggunakan CSS.
- Setiap bagian halaman diberi gaya untuk memastikan tampilan responsif dan menarik.
- Bagian hero diberi latar belakang hijau dengan teks berwarna putih, sementara fitur utama memiliki latar belakang putih dan bayangan lembut.

## Teknologi yang Digunakan

- **HTML**: Digunakan untuk struktur dan konten halaman.
- **CSS**: Digunakan untuk styling dan tata letak halaman.
